# Export: SodaCL

Converts the contract's schema and [quality rules](../quality-rules.md) into [SodaCL](https://docs.soda.io/soda-cl/soda-cl-overview.html) checks.

```bash
datacontract export sodacl datacontract.yaml --output checks.yml
```
